Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
выведет [77, 78, 79]. был создан срез от среза а, со второго символа(1) и до пятого(4) не включая его

```
